# 搜索标签页

macOS 风格的搜索起始页，支持 Google 和 Bing 切换。

## 功能

- 实时时间显示
- Google / Bing 搜索引擎切换（Logo 形式）
- 搜索框聚焦时果冻动画效果
- 毛玻璃 UI 风格

## 部署

```bash
npm install
npm run dev
```

### Vercel 部署

1. 推送到 GitHub
2. 在 [Vercel](https://vercel.com) 导入仓库
3. 自动识别 Next.js 并部署

或使用 CLI：

```bash
npx vercel
```

## 技术栈

- Next.js 14
- React 18
